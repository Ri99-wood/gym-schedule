初期設定が必要です

Firebase の設定を index.html に記入してください

1
Firebase Console でプロジェクトを作成
Firestore Database を作成 → テストモードで開始
2
プロジェクト設定 → マイアプリ → ウェブアプリを追加 → 設定をコピー
3
index.html の FIREBASE_CONFIG を書き換える： 
const FIREBASE_CONFIG = {
  apiKey: "実際のキー",
  authDomain: "xxx.firebaseapp.com",
  projectId: "xxx",
  storageBucket: "xxx.appspot.com",
  messagingSenderId: "111",
  appId: "1:111:web:xxx"
};
4
GitHub にプッシュ → Settings → Pages → ブランチを選んで保存
